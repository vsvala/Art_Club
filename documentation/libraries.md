# Libraries & Dependencies

## Frontend

### Core

| Library | Description |
|---|---|
| [React 18](https://react.dev/) | UI component library |
| [React DOM](https://react.dev/) | React renderer for the browser |
| [React Router v6](https://reactrouter.com/) | Client-side routing and navigation |

### State Management

| Library | Description |
|---|---|
| [Redux](https://redux.js.org/) | Predictable global state container |
| [React Redux](https://react-redux.js.org/) | React bindings for Redux |
| [Redux Thunk](https://github.com/reduxjs/redux-thunk) | Middleware for async Redux actions |

### UI & Styling

| Library | Description |
|---|---|
| [React Bootstrap](https://react-bootstrap.github.io/) | Bootstrap components built for React |
| [Bootstrap](https://getbootstrap.com/) | CSS framework |
| [React Datepicker](https://reactdatepicker.com/) | Accessible date picker component |
| [read-more-react](https://www.npmjs.com/package/read-more-react) | Expandable "read more" text component |

### Networking

| Library | Description |
|---|---|
| [Axios](https://axios-http.com/) | HTTP client for browser and Node.js |

### Testing

| Library | Description |
|---|---|
| [React Testing Library](https://testing-library.com/react) | Testing utilities for React components |
| [@testing-library/jest-dom](https://github.com/testing-library/jest-dom) | Custom Jest matchers for DOM assertions |
| [Cypress](https://www.cypress.io/) | End-to-end testing framework |

### Dev Tools

| Library | Description |
|---|---|
| [JSON Server](https://github.com/typicode/json-server) | Mock REST API for local development |
| [ESLint](https://eslint.org/) | JavaScript code linter |
| [Redux DevTools Extension](https://github.com/reduxjs/redux-devtools) | Browser extension for debugging Redux state |

## Backend

### Core

| Library | Description |
|---|---|
| [Express](https://expressjs.com/) | Minimal web framework for Node.js |
| [Nodemon](https://nodemon.io/) | Automatically restarts the server on file changes |

### Database

| Library | Description |
|---|---|
| [Mongoose](https://mongoosejs.com/) | MongoDB object modeling for Node.js |
| [mongoose-unique-validator](https://github.com/blakehaswell/mongoose-unique-validator) | Plugin for validating unique fields in Mongoose schemas |

### Authentication & Security

| Library | Description |
|---|---|
| [jsonwebtoken](https://github.com/auth0/node-jsonwebtoken) | Generates and verifies JSON Web Tokens |
| [bcrypt](https://github.com/kelektiv/node.bcrypt.js) | Password hashing library |
| [cors](https://github.com/expressjs/cors) | Middleware for enabling cross-origin requests |

### File Handling

| Library | Description |
|---|---|
| [Multer](https://github.com/expressjs/multer) | Middleware for handling multipart/form-data (image uploads) |

### Utilities

| Library | Description |
|---|---|
| [dotenv](https://github.com/motdotla/dotenv) | Loads environment variables from a `.env` file |
| [body-parser](https://github.com/expressjs/body-parser) | Parses incoming request bodies |
